# Bangumi UI

在 Seanime 侧栏查看 [Bangumi 番组计划](https://bgm.tv) 元数据。

## 安装

下载 `bangumi-ui.json` → 放入 `Seanime/extensions/` → Extensions → 重载。

## 使用

1. 打开番剧条目页，插件自动匹配 Bangumi 条目
2. 点左侧栏 **Bangumi** 查看数据

## 配置

API 端点如无法访问，在插件设置中改为：
```
https://api.bangumi.lol
```

## 开发

```bash
npm install
node build.js
```
